Mais que um sistema, uma filosofia.

Linux é o núcleo que move desde supercomputadores a smartphones. Criado por uma comunidade global, ele representa o poder do código aberto.

A História do Pinguim

1991: O Início de Tudo

Linus Torvalds, um estudante finlandês, anuncia seu projeto de um novo kernel de sistema operacional, inspirado no Minix.

1992: A Licença GPL e o GNU

O kernel Linux é licenciado sob a GNU General Public License (GPL), unindo-se às ferramentas do projeto GNU para formar um sistema operacional completo e livre.

1996: O Nascimento do Tux

O icônico mascote, o pinguim Tux, é escolhido para representar o Linux, consolidando sua identidade visual.

2004: O Ubuntu Chega

A Canonical lança o Ubuntu, uma distribuição focada em "Linux para seres humanos", que popularizou o sistema em desktops.

2007: O Domínio Móvel

O Google anuncia o Android, um sistema operacional para smartphones baseado no kernel Linux, que hoje domina o mercado mobile.

2012: Linux Conquista os Supercomputadores

O Linux se torna o sistema operacional dominante em supercomputação. Em novembro de 2012, mais de 94% dos 500 supercomputadores mais rápidos do mundo já rodavam Linux.

2014: Microsoft "Ama" o Linux

Em um movimento histórico, Satya Nadella, o novo CEO da Microsoft, declara que "a Microsoft ama o Linux". A empresa começa a integrar o Linux em seus produtos, como o Azure.

2018: WSL - Linux Dentro do Windows

A Microsoft lança o Windows Subsystem for Linux (WSL), permitindo que desenvolvedores executem um ambiente Linux completo diretamente no Windows 10.

Origem do Mascote

O pinguim Tux foi escolhido porque Linus Torvalds foi "mordido" por um pinguim em um zoológico e achou a experiência divertida. Ele queria um mascote com cara de "satisfeito".

🚀

Domínio Espacial

A SpaceX usa Linux em seus foguetes Falcon 9 e na cápsula Dragon. O sistema controla tudo, desde a telemetria até os sistemas de navegação, provando sua confiabilidade.

🎬

Estrela de Hollywood

Os efeitos visuais de filmes como "Titanic", "Avatar" e toda a saga da Marvel foram renderizados em fazendas de servidores rodando Linux.

💻

Criação do Git

Insatisfeito com os sistemas existentes, Linus Torvalds criou o Git em apenas duas semanas em 2005 para gerenciar o desenvolvimento do kernel Linux.
